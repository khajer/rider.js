# rider.js
fast way to create web-application

------------------

## Installation
npm install rider.js -g

## How to user

#### Create project 
``` 
rider create-app YOUAPP_NAME
```

```
cd YOUAPP_NAME
```

``` 
npm install 
```

#### Run Server
```
node index.js
```
